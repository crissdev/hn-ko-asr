# hn-ko-asr

Hacker News written in [Knockout](https://github.com/knockout/knockout/) using [abstract-state-router](https://github.com/TehShrike/abstract-state-router).


### Run the app

```sh
git clone https://github.com/crissdev/hn-ko-asr.git
npm install
npm start
```
